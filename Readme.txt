1. Для запуска проекта, необходимо наполнить базу данных test, в которую имеет доступ юзер root с паролем root.
Наполнение производится исполнением SQL-скрипта в корне проекта: start.sql
2. Соберите проект с помощью maven.
3. Сделайте деплой проекта на Tomcat.
4. Перейдите на страницу localhost:8080/users.
